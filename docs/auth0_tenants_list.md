# auth0 tenants list

List your tenants.

## Usage
```
auth0 tenants list [flags]
```

## Examples

```
  auth0 tenants list
  auth0 tenants ls
```




## Inherited Flags

```
      --debug           Enable debug mode.
      --no-color        Disable colors.
      --no-input        Disable interactivity.
      --tenant string   Specific tenant to use.
```
