# auth0 event-streams show

Display the name, type, status, subscriptions and other information about an event stream

## Usage
```
auth0 event-streams show [flags]
```

## Examples

```
  auth0 event-streams show
  auth0 event-streams show <event-id>
  auth0 event-streams show <event-id> --json
  auth0 event-streams show <event-id> --json-compact
```


## Flags

```
      --json           Output in json format.
      --json-compact   Output in compact json format.
```


## Inherited Flags

```
      --debug           Enable debug mode.
      --no-color        Disable colors.
      --no-input        Disable interactivity.
      --tenant string   Specific tenant to use.
```
