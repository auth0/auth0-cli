## auth0 apps use

Choose a default application for the Auth0 CLI

### Synopsis

Specify your preferred application for interaction with the Auth0 CLI.

```
auth0 apps use [flags]
```

### Examples

```
auth0 apps use <client-id>
```

### Options

```
  -h, --help   help for use
  -n, --none   Specify none of your apps.
```
