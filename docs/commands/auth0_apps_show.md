## auth0 apps show

Show an application.

```
auth0 apps show [flags]
```

### Examples

```
auth0 apps show 
auth0 apps show <id>
```

### Flags

```
  -h, --help     help for show
  -r, --reveal   Display the Client Secret as part of the command output.
```
