## auth0 apps list

List your existing applications. To create one try:
auth0 apps create

```
auth0 apps list [flags]
```

### Examples

```
auth0 apps list
auth0 apps ls
```

### Flags

```
  -h, --help     help for list
  -r, --reveal   Display the Client Secret as part of the command output.
```
