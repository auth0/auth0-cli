## auth0 roles list

List your roles

### Synopsis

List your existing roles. To create one try:
auth0 roles create

```
auth0 roles list [flags]
```

### Examples

```
auth0 roles list
auth0 roles ls
auth0 roles ls -n 100
```

### Options

```
  -h, --help         help for list
  -n, --number int   Number of apps to retrieve (default 50)
```
